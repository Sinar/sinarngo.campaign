sinarngo.campaign Installation
------------------------------

To install sinarngo.campaign using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sinarngo.campaign`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sinarngo.campaign

* Re-run buildout, e.g. with:

    $ ./bin/buildout
